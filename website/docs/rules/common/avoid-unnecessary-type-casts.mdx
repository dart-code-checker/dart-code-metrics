<RuleDetails version="4.7.0" severity="warning" />

Warns about of unnecessary use of casting operators.

### Example

```dart
class Example {
  final myList = <int>[1, 2, 3];

  void main() {
    final result = myList as List<int>; // LINT
  }
}
```
